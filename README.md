# Data Analysis Using R in Social Statistics

### This repository contains R-based data analysis methodologies from beginner to advanced levels.
### It includes R codes along with statistical explanations and interpretations.
  
You can find :
1) Basic R Coding : Basic syntax, Data preprocessing and manipulation, Visualization fundamentals (histograms, scatter plots, boxplots)
2) Descriptive Analysis - Summary statistics, Cross-tabulation, Correlation analysis, etc.
3) Inferential Analysis - Hypothesis testing, t-test, z-test, chi-squared test, F-test, ANOVA/MANOVA, Regression (linear, logistic), etc. 
4) Assumption & Parametric Tests - Normality test (e.g., Shapiro-Wilk test, Kolmogorov-Smirnov), Homogeneity of variance (Levene's test), etc. 
5) Model Diagnostics - Goodness-of-fit, Cook's distance, Residual analysis, Multicollinearity (VIF), etc.
6) Demension Reduction & Clustering - PCA, MCA, Factor analysis, K-means clustering, Hierarchical clustering, Cluster validation (shillouette score, elbow method), etc.
